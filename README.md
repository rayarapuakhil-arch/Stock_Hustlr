📈 Stock Exchange Management System

This project is a web-based Stock Exchange Management System (SEMS) that simulates real-time stock trading. It allows users to track live stock prices, manage portfolios, and perform virtual buy/sell operations, while providing admins with control over system data and analytics.

✨ Features

🔐 User Authentication (Login/Register with JWT)
👤 Role-based Access Control (User & Admin)
📊 Real-Time Stock Price Updates using WebSockets
💼 Portfolio Management (Holdings, Profit/Loss tracking)
💸 Virtual Trading System (Buy/Sell Stocks)
📈 Interactive Dashboard with Charts & Analytics
📋 Order History & Transaction Tracking
⚙️ User Settings (Dark Mode, Preferences)

🛠️ Tech Stack

Backend: Node.js, Express.js, Socket.io
Frontend: HTML, CSS (Tailwind/Bootstrap), JavaScript 
Database: MongoDB (Mongoose)
Authentication: JWT (JSON Web Tokens)
APIs: Alpha Vantage / Yahoo Finance (for stock data)

🧩 Architecture

The project follows a full-stack modular architecture:

Frontend: Handles UI, charts, and real-time updates
Backend: Manages APIs, authentication, and trading logic
Database: Stores users, portfolios, orders, and transactions
WebSockets: Enables live stock price updates
🗄️ Database Design
User: Stores account details and authentication
Portfolio: Tracks user investments and balance
Holding: Maintains stock quantities and values
Order: Records buy/sell transactions
Transaction: Logs fund activities
🚀 Key Functionalities
👤 Users can:
Register and log in securely
View real-time stock prices
Buy and sell stocks using virtual funds
Track portfolio performance (profit/loss)
View order history and transactions
🛠️ Admins can:
Monitor user activity
Manage system data
View reports and analytics
🎯 Objective

To build an interactive and beginner-friendly stock trading simulator that provides real-time insights, portfolio tracking, and hands-on learning for users without financial risk.
